Screen-cap

Screen capturing tool.

Download .zip Download .tar.gz View on GitHub

A simple screen capturing utility. Currently supports the following features -

  1. Allows you to capture screens every X seconds.
  2. Can be used for monitoring by capturing screens randomly once every X seconds.
  3. Supports two image formats - jpg and png
  4. Supports the ability to scale images to a certain percentage. This helps in reducing image size.
  5. Supports the ability to reduce the quality of the images.
